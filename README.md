# Diabetes Prediction Using Tongue Images

## Project Overview
This project is an AI-powered system that predicts the likelihood of diabetes using tongue images. The system applies image preprocessing and machine learning techniques to analyze tongue characteristics and provide a prediction. It aims to support early screening and assist healthcare professionals.

## Features
- Upload tongue images for analysis
- Image preprocessing and feature extraction
- Machine learning-based diabetes prediction
- User-friendly web interface
- Fast and accurate prediction

## Technologies Used
- Python
- Flask
- OpenCV
- NumPy
- Pandas
- Scikit-learn
- HTML
- CSS
- JavaScript

## Project Workflow
1. User uploads a tongue image.
2. The image is preprocessed to improve quality.
3. Features are extracted from the image.
4. The trained machine learning model predicts whether diabetes is likely.
5. The prediction result is displayed to the user.
